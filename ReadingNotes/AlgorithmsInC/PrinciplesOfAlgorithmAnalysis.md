# 常见的时间复杂度

| 时间复杂度 | 描述 |
| ---------- | ---- |
| 1          | 常数。程序的大部分指令只执行一次或至多执行几次。 |
| logN       | 对数。将一个大问题转化为一系列小问题，每一步都把问题规模分解为恒定的小部分。 |
| N          | 线性。对每个输入元素都进行处理。 |
| NlogN      | NlogN。将问题分解成较小的子问题，独立求解每个子问题，然后综合子问题的解得到原问题的解。 |
| $$N^2$$        | 二次方。一般存在于双重嵌套的循环中。 |
| $$N^3$$ | 三次方。一般存在于三重循环中。 |
| $$2^N$$ | 指数。常见于蛮力法解决问题的过程中。 |
